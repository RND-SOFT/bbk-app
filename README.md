# BBK::App

Classes for building services based on BBK stack.

## Installation

Adding to a gem:

```ruby
# my-cool-gem.gemspec

Gem::Specification.new do |spec|
  # ...
  spec.add_dependency "bbk-app", "~> 1.0.0"
  # ...
end
```

Or adding to your project:

```ruby
# Gemfile

gem "bbk-app", "~> 1.0.0"
```

### Supported Ruby versions

* Ruby (MRI) >= 2.5.0

### Tested Ruby versions

* Ruby (MRI) 2.5.x
* Ruby (MRI) 3.0.x
